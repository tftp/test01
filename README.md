## **Запуск проекта**
Для запуска проекта необходимо запустить в папке проекта команду:
`docker-compose up -d`

## **Проверка результата**
Ввести команду `curl http://localhost`
На экран выведет результат: *Hello from Effective Mobile!*

## **Как работает схема nginx-backend**
Backend - http сервер, который слушает 8080 порт и выдает всем приходящим сообщение: *Hello from Effective Mobile!*
Nginx - web сервер, который работает в режиме реверс прокси, который слушает 80 порт и перенаправляет на 8080 порт нашего backend приложения
